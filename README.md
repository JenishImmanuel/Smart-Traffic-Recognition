# Traffic Sign Recognition Project

## Overview
This project uses machine learning to classify traffic signs based on image inputs. It employs a convolutional neural network (CNN) trained on a diverse dataset of road signs to improve accuracy and ensure reliable predictions.

## Features
- **Preprocessing**: Image normalization and augmentation for better model generalization.
- **CNN-based Model**: Built using TensorFlow/Keras for high-accuracy sign recognition.
- **Training & Evaluation**: Model tested on a dataset of traffic signs with high confidence scores.
- **Automated Predictions**: Classifies images into predefined categories with a confidence score.
